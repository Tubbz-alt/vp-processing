1 Getting started

Note: This documentation assumes you are working in RStudio.

1.1 Install packages

In this tutorial we use a number of R packages, which you need to install first:

install.packages(c(
  "devtools",  # To install packages from GitHub
  "dplyr",     # To process dataframes
  "lubridate", # To process dates
  "bookdown"   # To generate this documentation
))
devtools::install_github("adokter/bioRad", ref="master") # To process vp data

1.2 Download the code

To use and especially adapt the code of this tutorial, fork or clone this repository: https://github.com/enram/timamp-etl

1.3 Edit the code

Once downloaded, go to the /src directory:

  1. Open timamp-etl.Rproj as a project in RStudio
  2. Open one of the .Rmd files in RStudio to see and edit the code.
  3. Test your code by running it.

1.4 Generating fancy documentation ðŸ“•

What your are reading here (in your browser) is all generated from the .Rmd files in the src directory, using the R package bookdown.

In fact, if you notice a typo in this documentation, you can correct it by clicking the pencil icon in the menu at the top, which will take you straight to the correct .Rmd file in the original repository on GitHub. If you click the pencil icon on that GitHub page, you will be able to edit right away or create a copy (a fork) of the repository to do so, depending on your access rights.

If you want to generate your own fancy documentation from the adapted .Rmd files, then:

  1. Go to Build pane in RStudio
  2. Click the Build book button
  3. Wait for the build to finish: that might take a while as it will run the code from all the .Rmd files.
  4. Go to the docs repository
  5. Open index.html in your browser
  6. See your updated documentation
